📱 Project Overview
Taskplan is a mobile task management app designed to help users organize their daily tasks efficiently. The design focuses on clean layouts, intuitive user flows, and consistent visual components across all screens.
---
🖥️ Screens Designed
Screen	Description
Home Page	Dashboard showing today's progress summary, active and completed tasks
New Task	Form screen to create a new task with project selection, date and time picker
Categories	Filter and inbox view to organize tasks by Personal, Work, Details, and Inbox
Settings	Account, privacy, security, billing, and notification settings
---
🎨 Design Highlights
User-Centered Design — Layouts built around real user needs and task workflows
Consistent Components — Reusable buttons, cards, navigation bars, and filter chips
Visual Hierarchy — Clear typography, spacing, and color usage to guide user attention
Color Palette — Clean blue and white theme with soft background tones (F5F5F5)
Mobile-First — Designed for mobile screen dimensions with proper padding and touch targets
